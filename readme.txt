1. Open shell - type cmd in search box
2. Type 
pip install streamlit
streamlit hello

The streamlit app will appear in the browser at localhost:8501

3. Open Notepad and enter your python code into the file app.py

4. In the shell type:

streamlit run C:\Users\Joyce\Desktop\Data_Science\Streamlit\app.py

5. Update your file, save and then refresh in the browser
6. Type Ctrl and C in the shell to stop the app running and exit
7. Upload file to github repository
8. Send request to Streamlit to host app from github repository


To deploy an app - read the following:
https://docs.streamlit.io/en/stable/deploy_streamlit_app.html
